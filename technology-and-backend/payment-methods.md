# Payment methods

Depending on the platform environment (web or mobile app), users can complete transactions using $MESI tokens or through in-app payments (IAP).

## Available Payment Methods

The table below outlines the payment methods for various app functions:

|                      | Mobile app         | Web app                 |
| -------------------- | ------------------ | ----------------------- |
| Bid to meet          | IAP                | Token                   |
| Buy/sell NFT         | IAP                | Token                   |
| Mesi Live Tipping    | IAP                | Token                   |
| Private Message      | IAP                | Token                   |
| Subscribe            | IAP                | Token                   |
| Affiliate Marketing  | Google / Apple Pay | Token                   |
| AI Premium Functions | IAP                | Token                   |
| Buy $MESI            | IAP                | Credit card / Bank wire |

Regardless of the payment method, all withdrawals from the Mesi platform are made in $MESI tokens.<br>
